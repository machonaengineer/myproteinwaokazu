# myprotein-site SEO package

GitHub Pagesにそのままアップロードできる静的HTML一式です。

## 使い方

1. このZIPを解凍
2. 中身を `myprotein-site` リポジトリ直下にアップロード
3. 既存ファイルは上書き
4. Commit changes
5. 数分後に `https://machonaengineer.github.io/myprotein-site/` を確認

## 重要ファイル

- index.html: トップ・収益ハブ
- myprotein-sale.html: 「マイプロテイン セール」狙い
- myprotein-review.html: 「マイプロテイン」「口コミ」狙い
- myprotein-osusume.html: 「マイプロテイン おすすめ」狙い
- myprotein-aji.html: 「マイプロテイン 味 おすすめ」狙い
- myprotein-shoshinsha.html: 「マイプロテイン 初心者」狙い
- myprotein-creatine.html: 「マイプロテイン クレアチン」狙い
- eaa-bcaa.html: 「EAA BCAA 違い」狙い
- protein-osusume.html: 「プロテイン おすすめ」狙い
- impact-whey.html: 「Impact ホエイ」狙い
- myprotein-diet.html: 減量系
- myprotein-clear-whey.html: クリアホエイ系
- privacy.html/contact.html: 信頼性補強
- robots.txt/sitemap.xml: クロール補助
- style.css: 共通デザイン

## アップロード後

Search Consoleで `https://machonaengineer.github.io/myprotein-site/sitemap.xml` を送信してください。
